# Java Console Calculator

A simple Java-based console calculator built for the Java Developer Internship task.

## Features
- Uses Scanner for user input
- Supports:
  - Addition
  - Subtraction
  - Multiplication
  - Division (with divide-by-zero handling)
- Uses loops for repeated calculations
- Uses separate methods for each operation

## How to Run
1. Install Java JDK
2. Open Terminal
3. Compile:
   javac Calculator.java
4. Run:
   java Calculator
